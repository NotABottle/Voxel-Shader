# Voxel-Shader
A real-time mesh voxelization shader made in Unity 6

### Showcase Footage
https://github.com/user-attachments/assets/2350f2f1-9d6c-4beb-9a6a-b9fd53912bd3

### Description
Takes the provided mesh and generates a grid the size of the mesh bounds at the user-defined resolution. Points then check to see whether they collide with the mesh and if they do then a voxel is rendered at that point.
